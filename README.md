# Task Manager App

Mobile task management application built with React Native and Expo.

## Features

- Create tasks
- View task details
- Update task status
- Delete tasks
- Sort tasks by date or status
- Local data persistence with AsyncStorage
- Date validation and formatted input

## Tech Stack

- React Native
- Expo
- TypeScript
- React Navigation
- AsyncStorage
- date-fns

## Installation

```bash
npm install
npx expo start
```
